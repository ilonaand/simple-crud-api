# simple-crud-api

## Feature list

- Serve API with auto routing
- API tests 

## Usage

- You need 18 LTS version of Node.js
- Fork and clone this repository (optionally subscribe to repo changes)
- Run `npm i` to install dependencies 
- The application is run in development mode using nodemon  ` npm run start:dev` and stop with Ctrl+C
- The application is run in production mode ` npm run start:prod` and stop with Ctrl+C
- Tests are run  `npm run test` and stop with Ctrl+C

## Enable API 
- `api/users` for GET all users and POST user
- `api/user/${userId}` for GET, PUT, DELETE user
